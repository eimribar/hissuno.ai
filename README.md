# Fin.ai Pixel-Perfect Replica

This is a complete pixel-perfect replica of the Fin.ai homepage (https://fin.ai/).

## What's Included

- **Complete HTML**: The full HTML structure with all elements, classes, and data attributes preserved
- **8 CSS Files**: All stylesheets including custom fonts, layouts, and animations
- **38 JavaScript Files**: All interactive functionality and framework code
- **32 Images**: All graphics, logos, thumbnails, and SVG assets
- **14 Custom Fonts**: All WOFF2 font files for typography

## Directory Structure

```
fin-replica/
├── index.html          # Main HTML file
├── assets/

│   ├── js/            # 38 JavaScript files
│   ├── images/        # 32 image files (JPEG, PNG, SVG)

```

## How to View

Simply open `index.html` in any modern web browser:

```bash
open index.html
```

Or double-click the `index.html` file from Finder.

## Technical Details

- **Framework**: Next.js (based on meta tags and file structure)
- **Styling**: Custom CSS with Tailwind-inspired utility classes
- **Fonts**: Custom WOFF2 fonts loaded from local assets
- **Images**: Optimized JPEG and SVG files
- **Browser Support**: Modern browsers (Chrome, Firefox, Safari, Edge)

## Asset Paths

All assets are referenced with relative paths:
- CSS: `./assets/css/`
- JavaScript: `./assets/js/`
- Images: `./assets/images/`
- Fonts: `./assets/fonts/`

## Notes

This is a static replica meant for development and design reference purposes. The original site is powered by Next.js with server-side functionality that is not replicated here.

All assets have been properly organized and paths updated to work as a standalone local site.
